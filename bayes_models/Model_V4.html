Model V4 — Hierarchical Cost-Per-Person Baseline

Model V4 extends V3 by placing a hierarchical prior on the region-specific intercept \(\alpha_i\):

\[\alpha_i \sim \mathcal{N}(\mu_\lambda \cdot c_i,\; \tau_\alpha)\]

where \(c_i\) is the per-person HR cost for region \(i\) (known constant from hr_cost_per_person.csv), \(\mu_\lambda\) is a global cost coefficient estimated from the data, and \(\tau_\alpha\) captures residual region-level variation not explained by cost.

This is identifiable (unlike placing \(c_i \cdot \lambda_i\) directly in the mean alongside a free intercept): \(\mu_\lambda\) measures the global cost–burden relationship while \(\alpha_i\) retains the flexibility to deviate from the cost-scaled expectation.

Model V4 Specification

Mean function (same as V3): \[\mu_{i,t} = \alpha_i + \beta_i\cos\!\Big(\frac{2\pi t}{52}\Big) + \gamma_i\sin\!\Big(\frac{2\pi t}{52}\Big) + \delta_{\text{pre},i}\,\mathbb{1}(t\bmod 52=0) + \delta_{\text{mid},i}\,\mathbb{1}(t\bmod 52=1) + \delta_{\text{post},i}\,\mathbb{1}(t\bmod 52=2) + \sigma_{\text{pre}}\,\mathbb{1}(t=86,\,i=\text{MW}) + \sigma_{\text{mid}}\,\mathbb{1}(t=87,\,i=\text{MW}) + \sigma_{\text{post}}\,\mathbb{1}(t=88,\,i=\text{MW})\]

Hierarchical prior (V4 key change): \[\alpha_i \sim \mathcal{N}(\mu_\lambda \cdot c_i,\; \tau_\alpha)\]

model <-
"
model{
  for(i in 1:I){
    #------------
    # Likelihood
    #------------
    y[i,1] ~ dnorm(mu[i,1], tau[i])
    for(t in 2:T){
      y[i,t] ~ dnorm(mu[i,t] + (phi * (y[i,t-1] - mu[i,t-1])), tau[i])
    }

    # Mean: alpha[i] has hierarchical prior centred on mu_lambda * c_i
    for(t in 1:T){
      mu[i,t] <- alpha[i] +
                 beta[i]       * cos((2 * pi) * (t/52)) +
                 gamma[i]      * sin((2 * pi) * (t/52)) +
                 delta_pre[i]  * ny_pre[t]  +
                 delta_mid[i]  * ny_mid[t]  +
                 delta_post[i] * ny_post[t] +
                 sigma_pre     * fr_pre[t]  * mw[i] +
                 sigma_mid     * fr_mid[t]  * mw[i] +
                 sigma_post    * fr_post[t] * mw[i]
    }

    alpha[i] <- mu_lambda * cost_per_person[i]
    beta[i]   ~ dnorm(0, 0.001)
    gamma[i]  ~ dnorm(0, 0.001)
    tau[i]    ~ dgamma(0.001, 0.001)
    delta_pre[i]  ~ dnorm(0, 0.001)
    delta_mid[i]  ~ dnorm(0, 0.001)
    delta_post[i] ~ dnorm(0, 0.001)
  }
  phi        ~ dunif(-1, 1)
  sigma_pre  ~ dnorm(0, 0.001)
  sigma_mid  ~ dnorm(0, 0.001)
  sigma_post ~ dnorm(0, 0.001)
  # Hyperpriors
  mu_lambda  ~ dnorm(0, 0.001)
}
"
